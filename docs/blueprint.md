# **App Name**: SoulSync

## Core Features:

- Psychological Onboarding: Guide new users through a personalized personality assessment including Big Five (OCEAN), MBTI, zodiac traits, and custom trait questions.
- Psych-Match Feed: Curated recommendation feed displaying profiles of psychologically compatible users, sorted and emphasized with compatibility scores
- Explore All Users: Allow users to browse all profiles on the platform with compatibility scores displayed.
- Chat: Allow matched users to communicate directly through text messages within the platform.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke a sense of calm and emotional intelligence.
- Background color: Light gray (#F0F0F0), a very desaturated lavender, for a clean and unobtrusive backdrop.
- Accent color: Muted violet (#A9A9D1), analogous to lavender, used to highlight important interactive elements without overwhelming the design.
- Body and headline font: 'Alegreya', a serif font to enhance readability and imbue an elegant, intellectual feel, aligning with the depth of psychological insights.
- Use recognizable icons for MBTI types and zodiac signs.
- Prioritize clarity and warmth in the UI, using card-based designs to emphasize compatibility scores and psychological summaries prominently.
- Subtle animations during the generation of the compatibility report.
